# <a name="design-details-variance"></a><span data-ttu-id="de00f-103">Designoplysninger: Afvigelse</span><span class="sxs-lookup"><span data-stu-id="de00f-103">Design Details: Variance</span></span>
<span data-ttu-id="de00f-104">Afvigelsen er defineret som forskellen mellem den faktiske kostpris og standardkostprisen, som beskrevet i følgende formel.</span><span class="sxs-lookup"><span data-stu-id="de00f-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="de00f-105">faktiske omkostninger – standardomkostninger = afvigelse</span><span class="sxs-lookup"><span data-stu-id="de00f-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="de00f-106">Hvis de faktiske omkostninger ændres, eksempelvis fordi du bogfører et varegebyr på en senere dato, opdateres afvigelsen tilsvarende.</span><span class="sxs-lookup"><span data-stu-id="de00f-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="de00f-107">Værdiregulering påvirker ikke beregningen af afvigelse, fordi værdiregulering kun ændrer lagerværdien.</span><span class="sxs-lookup"><span data-stu-id="de00f-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>
