   <span data-ttu-id="cd18f-116">Vinduet **Valg af rapportlayout** viser en liste over alle de rapporter, der er tilgængelige i det firma, der er angivet i feltet Virksomhed øverst i vinduet.</span><span class="sxs-lookup"><span data-stu-id="cd18f-116">The** Report Layout Selection** window lists all the reports that are available in the company that is specified in the Company field at the top of the window.</span></span>

4. <span data-ttu-id="cd18f-120">Hvis du vil oprette en kopi af et eksisterende brugerdefineret layout, skal du vælge det eksisterende brugerdefinerede layout på listen og derefter vælge **Kopiér**.</span><span class="sxs-lookup"><span data-stu-id="cd18f-120">If you want to create a copy of an existing custom layout, select the existing custom layout in the list, and then choose **Copy**.</span></span>

   2. <span data-ttu-id="cd18f-126">Hvis du vil tilføje en brugerdefineret Word-rapportlayouttype, skal du markere afkrydsningsfeltet **Indsæt Word-layout**.</span><span class="sxs-lookup"><span data-stu-id="cd18f-126">To add a custom Word report layout type, then select the **Insert Word Layout** check box.</span></span>
   3. <span data-ttu-id="cd18f-127">Hvis du vil tilføje en brugerdefineret RDLC-rapportlayouttype, skal du markere afkrydsningsfeltet **Indsæt RDLC-layout**.</span><span class="sxs-lookup"><span data-stu-id="cd18f-127">To add a custom RDLC report layout type, then select the **Insert RDLC Layout** check box.</span></span>

      <span data-ttu-id="cd18f-129">De nye brugerdefinerede layout vises i vinduet **Tilpassede rapportlayouts**.</span><span class="sxs-lookup"><span data-stu-id="cd18f-129">The new custom layouts appear in the **Custom Report Layouts** window.</span></span> <span data-ttu-id="cd18f-130">Hvis et nyt layout er baseret på et indbygget layout, så har den ordene **Kopi af et indbygget layout** i feltet **Beskrivelse**.</span><span class="sxs-lookup"><span data-stu-id="cd18f-130">If a new layout is based on a built-in layout, then it has the words **Copy of a Built-in Layout** in the **Description** field.</span></span> <span data-ttu-id="cd18f-131">Hvis der ikke var noget indbygget layout for rapporten, har det nye layout ordene **Nyt layout** i feltet **Beskrivelse**, fordi det brugerdefinerede layout er tomt.</span><span class="sxs-lookup"><span data-stu-id="cd18f-131">If there was no built-in layout for the report, then the new layout has the words **New Layout** in the **Description** field, which indicates that custom layout is blank.</span></span>
6. <span data-ttu-id="cd18f-132">Som standard er feltet **Virksomhedsnavn** tomt, hvilket betyder, at det brugerdefinerede layout bliver tilgængeligt for rapporten i alle firmaer.</span><span class="sxs-lookup"><span data-stu-id="cd18f-132">By default, the **Company Name** field is blank, which means that the custom layout will be available for the report in all companies.</span></span> <span data-ttu-id="cd18f-133">Hvis du kun vil gøre det brugerdefinerede layout tilgængeligt i en bestemt virksomhed, skal du vælge **Rediger** og derefter indstille feltet **Virksomhedsnavn** til den ønskede virksomhed.</span><span class="sxs-lookup"><span data-stu-id="cd18f-133">To make the custom layout available in a specific company only, choose **Edit**, and then set the **Company Name** field to the company that you want.</span></span>
