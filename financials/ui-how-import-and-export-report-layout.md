# <a name="how-to-import-and-export-a-report-or-document-layout"></a><span data-ttu-id="ee345-103">Fremgangsmåde: Importere og eksportere et rapport- eller dokumentlayout</span><span class="sxs-lookup"><span data-stu-id="ee345-103">How to: Import and Export a Report or Document Layout</span></span>
<span data-ttu-id="ee345-104">Du kan importere og eksportere et eksisterende brugerdefineret rapportlayout som en fil til og fra en placering på din computer og netværket.</span><span class="sxs-lookup"><span data-stu-id="ee345-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="ee345-105">For eksempel kan du eksportere et rapportlayout og derefter sende filen til en anden person for redigering.</span><span class="sxs-lookup"><span data-stu-id="ee345-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="ee345-106">Denne person kan derefter foretage ændringer af layoutet og returnere filen til dig, så du kan importere det tilbage.</span><span class="sxs-lookup"><span data-stu-id="ee345-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  
  
> [!IMPORTANT]  
>  <span data-ttu-id="ee345-107">Du kan ikke importere eller eksportere indbyggede rapportlayout.</span><span class="sxs-lookup"><span data-stu-id="ee345-107">You cannot import or export built-in report layouts.</span></span>  
  
### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="ee345-108">Sådan eksporterer du et rapportlayout til en fil</span><span class="sxs-lookup"><span data-stu-id="ee345-108">To export a report layout to a file</span></span>  
  
1.  <span data-ttu-id="ee345-109">Vælg ikonet ![Søg efter side eller rapport](media/ui-search/search_small.png "Ikonet Søg efter side eller rapport"), angiv **Valg af rapportlayout**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="ee345-109">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
2.  <span data-ttu-id="ee345-110">Vælg rækken med den rapport, der indeholder det brugerdefinerede rapportlayout, du vil eksportere, og brug derefter fanen **Startside** til at vælge **Brugerdefinerede layout** i gruppen **Proces**.</span><span class="sxs-lookup"><span data-stu-id="ee345-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
3.  <span data-ttu-id="ee345-111">I vinduet **Rapportlayout** skal du markere det rapportlayout, du vil eksportere til en fil, og derefter vælge **Eksportér layout** under fanen **Startside** i gruppen **Proces**.</span><span class="sxs-lookup"><span data-stu-id="ee345-111">In the **Report Layouts** window, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  
  
4.  <span data-ttu-id="ee345-112">Brug dialogboksen **Udlæs fil** til at vælge **Gem**, og gem derefter filen til en placering på din computer eller netværket.</span><span class="sxs-lookup"><span data-stu-id="ee345-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  
  
### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="ee345-113">Sådan importerer du en rapportlayoutfil</span><span class="sxs-lookup"><span data-stu-id="ee345-113">To import a report layout file</span></span>  
  
1.  <span data-ttu-id="ee345-114">Sørg for, at den relevante fil, der definerer rapportens layout, er tilgængelig på din computer eller netværket.</span><span class="sxs-lookup"><span data-stu-id="ee345-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  
  
     <span data-ttu-id="ee345-115">En Word-layoutfil skal have filtypenavnet .docx .</span><span class="sxs-lookup"><span data-stu-id="ee345-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="ee345-116">En RDLC-rapportlayoutfil skal have filtypenavnet .rdlc eller .rdl.</span><span class="sxs-lookup"><span data-stu-id="ee345-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  
  
2.  <span data-ttu-id="ee345-117">Vælg ikonet ![Søg efter side eller rapport](media/ui-search/search_small.png "Ikonet Søg efter side eller rapport"), angiv **Valg af rapportlayout**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="ee345-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
3.  <span data-ttu-id="ee345-118">Vælg rækken med den rapport, du vil importere rapportlayoutet til, og brug derefter fanen **Startside** til at vælge **Brugerdefinerede layout** i gruppen **Proces**.</span><span class="sxs-lookup"><span data-stu-id="ee345-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
4.  <span data-ttu-id="ee345-119">I vinduet **Rapportlayout** skal du markere det rapportlayout, du vil importere filen til, og derefter vælge **Importér layout** under fanen **Startside** i gruppen **Proces**.</span><span class="sxs-lookup"><span data-stu-id="ee345-119">In the **Report Layouts** window, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  
  
5.  <span data-ttu-id="ee345-120">Brug dialogboksen **Importér** til at vælge det dokument, der definerer rapportlayoutet, og vælg derefter **Åbn**.</span><span class="sxs-lookup"><span data-stu-id="ee345-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>  
  
 <span data-ttu-id="ee345-121">Det oprindelige brugerdefinerede rapportlayout erstattes med det importerede rapportlayout.</span><span class="sxs-lookup"><span data-stu-id="ee345-121">The original custom report layout is replaced with the imported report layout.</span></span>
