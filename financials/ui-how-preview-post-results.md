# <a name="how-to-preview-posting-results"></a><span data-ttu-id="e8f03-103">Fremgangsmåde: Vise bogføringsresulterer</span><span class="sxs-lookup"><span data-stu-id="e8f03-103">How to: Preview Posting Results</span></span>
<span data-ttu-id="e8f03-104">På hvert dokument og kladde, som kan bogføres, kan du vælge knappen **Vis bogføring** for at se de forskellige typer poster, der oprettes, når du bogfører dokumentet eller kladden.</span><span class="sxs-lookup"><span data-stu-id="e8f03-104">On every document and journal that can be posted, you can choose the **Preview Posting** button to review the different types of entries that will be created when you post the document or journal.</span></span>

## <a name="to-preview-gl-entries-that-will-result-from-posting-a-purchase-invoice"></a><span data-ttu-id="e8f03-105">Sådan vises finansposter, der stammer fra bogføring af en købsfaktura</span><span class="sxs-lookup"><span data-stu-id="e8f03-105">To preview G/L entries that will result from posting a purchase invoice</span></span>
1. <span data-ttu-id="e8f03-106">Vælg ikonet ![Søg efter side eller rapport](media/ui-search/search_small.png "Ikonet Søg efter side eller rapport"), angiv **Købsfakturaer**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="e8f03-106">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Purchase Invoices**, and then choose the related link.</span></span>
2. <span data-ttu-id="e8f03-107">Opret en købsfaktura.</span><span class="sxs-lookup"><span data-stu-id="e8f03-107">Create a purchase invoice.</span></span> <span data-ttu-id="e8f03-108">Du kan finde flere oplysninger under [Fremgangsmåde: Registrere køb](purchasing-how-record-purchases.md).</span><span class="sxs-lookup"><span data-stu-id="e8f03-108">For more information, see [How to: Record Purchases](purchasing-how-record-purchases.md).</span></span>
3. <span data-ttu-id="e8f03-109">Vælg **Vis bogføring**.</span><span class="sxs-lookup"><span data-stu-id="e8f03-109">Choose **Preview Posting**.</span></span>

   <span data-ttu-id="e8f03-111">Vinduet **Vis finansposter** viser hvilke poster der oprettes, når du bogfører købsfakturaen.</span><span class="sxs-lookup"><span data-stu-id="e8f03-111">The **G/L Entries Preview** window shows which entries will be created when you post the purchase invoice.</span></span>
