### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="aa16d-108">Sådan eksporterer du et rapportlayout til en fil</span><span class="sxs-lookup"><span data-stu-id="aa16d-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="aa16d-109">Vælg ikonet ![Elpære, der åbner funktionen Fortæl mig](media/ui-search/search_small.png "Fortæl mig, hvad du vil foretage dig"), angiv **Valg af rapportlayout**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="aa16d-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="aa16d-110">Vælg rækken med den rapport, der indeholder det brugerdefinerede rapportlayout, du vil eksportere, og vælg derefter handlingen **Brugerdefinerede layout**.</span><span class="sxs-lookup"><span data-stu-id="aa16d-110">Select the row for the report that contains the custom report layout that you want to export, and then choose the **Custom Layouts** action.</span></span>  

3.  <span data-ttu-id="aa16d-111">På siden **Rapportlayout** skal du markere det rapportlayout, du vil eksportere til en fil, og derefter vælge handlingen **Eksportér layout**.</span><span class="sxs-lookup"><span data-stu-id="aa16d-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then choose the **Export Layout** action.</span></span>  

4.  <span data-ttu-id="aa16d-112">Brug dialogboksen **Udlæs fil** til at vælge knappen **Gem**, og gem derefter filen til en placering på din computer eller netværket.</span><span class="sxs-lookup"><span data-stu-id="aa16d-112">In the **Export File** dialog box, choose the **Save** button, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="aa16d-113">Sådan importerer du en rapportlayoutfil</span><span class="sxs-lookup"><span data-stu-id="aa16d-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="aa16d-114">Sørg for, at den relevante fil, der definerer rapportens layout, er tilgængelig på din computer eller netværket.</span><span class="sxs-lookup"><span data-stu-id="aa16d-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="aa16d-115">En Word-layoutfil skal have filtypenavnet .docx .</span><span class="sxs-lookup"><span data-stu-id="aa16d-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="aa16d-116">En RDLC-rapportlayoutfil skal have filtypenavnet .rdlc eller .rdl.</span><span class="sxs-lookup"><span data-stu-id="aa16d-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="aa16d-117">Vælg ikonet ![Elpære, der åbner funktionen Fortæl mig](media/ui-search/search_small.png "Fortæl mig, hvad du vil foretage dig"), angiv **Valg af rapportlayout**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="aa16d-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="aa16d-118">Vælg rækken med den rapport, du vil importere rapportlayoutet i, og vælg derefter handlingen **Brugerdefinerede layout**.</span><span class="sxs-lookup"><span data-stu-id="aa16d-118">Select the row for the report to which you want to import the report layout, and then choose the **Custom Layouts** action.</span></span>  

4.  <span data-ttu-id="aa16d-119">På siden **Rapportlayout** skal du markere det rapportlayout, du vil importere filen i, og derefter vælge handlingen **Importér layout**.</span><span class="sxs-lookup"><span data-stu-id="aa16d-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then choose the **Import Layout** action.</span></span>  

5.  <span data-ttu-id="aa16d-120">Brug dialogboksen **Importér** til at vælge det dokument, der definerer rapportlayoutet, og vælg derefter knappen **Åbn**.</span><span class="sxs-lookup"><span data-stu-id="aa16d-120">In the **Import** dialog box, select the document that defines the report layout, and then choose the **Open** button.</span></span>  

 <span data-ttu-id="aa16d-121">Det oprindelige brugerdefinerede rapportlayout erstattes med det importerede rapportlayout.</span><span class="sxs-lookup"><span data-stu-id="aa16d-121">The original custom report layout is replaced with the imported report layout.</span></span>
