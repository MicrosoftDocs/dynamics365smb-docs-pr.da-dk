# <a name="create-bin-contents"></a><span data-ttu-id="71f94-104">Oprette placeringsindhold</span><span class="sxs-lookup"><span data-stu-id="71f94-104">Create Bin Contents</span></span>
<span data-ttu-id="71f94-105">Når du har oprettet placeringerne, kan du oprette placeringsindholdet.</span><span class="sxs-lookup"><span data-stu-id="71f94-105">After you have set up your bins, you can set up the bin contents.</span></span> <span data-ttu-id="71f94-106">Du kan med andre ord sætte de varer op, som du vil opbevare på en bestemt placering, og angive regler, der styrer anbringelse af en bestemt vare på placeringen.</span><span class="sxs-lookup"><span data-stu-id="71f94-106">In other words, you can set up the items you want to store in any given bin and set the rules that govern filling the bin with a particular item.</span></span> <span data-ttu-id="71f94-107">Du kan gøre dette manuelt på siden **Placeringsindhold** eller automatisk med siden **Opret placeringsindholdskladde**.</span><span class="sxs-lookup"><span data-stu-id="71f94-107">You can do this manually on the **Bin Contents** page or automatically with the **Create Bin Content Worksheet** page.</span></span>

## <a name="to-create-bin-content-manually"></a><span data-ttu-id="71f94-108">Sådan oprettes placeringsindhold manuelt</span><span class="sxs-lookup"><span data-stu-id="71f94-108">To create bin content manually</span></span>  
1.  <span data-ttu-id="71f94-109">Vælg ikonet ![Elpære, der åbner funktionen Fortæl mig](media/ui-search/search_small.png "Fortæl mig, hvad du vil foretage dig"), angiv **Lokationer**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="71f94-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Locations**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="71f94-110">Marker den lokation, hvor du vil konfigurere placeringsindhold, og vælg derefter handlingen **Placeringer**.</span><span class="sxs-lookup"><span data-stu-id="71f94-110">Select the location where you want to set up bin contents,  and then choose the **Bins** action.</span></span>  
3.  <span data-ttu-id="71f94-111">Marker den placering, hvor du vil konfigurere indholdet, og vælg derefter handlingen **Indhold**.</span><span class="sxs-lookup"><span data-stu-id="71f94-111">Select the bin where you want to set up contents, and then choose the **Contents** action.</span></span>  
4.  <span data-ttu-id="71f94-112">For hvert element, du vil gemme i placeringen, skal du udfylde en linje på siden **Placeringsindhold** med de relevante oplysninger.</span><span class="sxs-lookup"><span data-stu-id="71f94-112">For each item that you want to store in the bin, fill in a line on the **Bin Contents** page with the appropriate information.</span></span> <span data-ttu-id="71f94-113">Nogle felter er allerede udfyldt med oplysninger om placeringen.</span><span class="sxs-lookup"><span data-stu-id="71f94-113">Some of the fields are filled in already with information about the bin.</span></span>  
5.  <span data-ttu-id="71f94-114">Udfyld først feltet **Varenr.**, og – hvis du bruger styret læg-på-lager og pluk – udfyld derefter de andre felter som f.eks. feltet **Enhedskode**, felterne **Maks. antal** og **Min. antal**.</span><span class="sxs-lookup"><span data-stu-id="71f94-114">First fill in the **Item No.** field, and then, if you are using directed put-away and pick, fill in the other fields such as the **Unit of Measure Code**, **Max. Qty.**, and **Min. Qty.** fields.</span></span>  

<span data-ttu-id="71f94-115">Vælg feltet **Fast**, hvis det er nødvendigt.</span><span class="sxs-lookup"><span data-stu-id="71f94-115">Select the **Fixed** field if necessary.</span></span> <span data-ttu-id="71f94-116">Hvis placeringen skal fungere som standardplacering for varen, skal du markere feltet **Standardplacering**.</span><span class="sxs-lookup"><span data-stu-id="71f94-116">If the bin is to be used as the default bin for the item, select the **Default Bin** field.</span></span>  

<span data-ttu-id="71f94-117">Hvis du bruger styret læg-på-lager og pluk, og du har angivet de korrekte dimensionsoplysninger for hver vares måleenhed, kontrolleres det maksimale antal, du har angivet på siden **Placeringsindhold**, i forhold til placeringens fysiske kapacitet.</span><span class="sxs-lookup"><span data-stu-id="71f94-117">If you are using directed put-away and pick, and if you have entered the correct dimensional information on the item card about each item’s units of measure, the maximum quantity that you enter on the **Bin Contents** page is verified against the physical capabilities of the bin.</span></span> <span data-ttu-id="71f94-118">Herefter benyttes minimums- og maksimumsantallet, når der beregnes genopfyldning, og der foreslås læg-på-lager-aktiviteter.</span><span class="sxs-lookup"><span data-stu-id="71f94-118">The minimum and maximum quantities are used when calculating bin replenishment and suggested put-aways.</span></span>  

<span data-ttu-id="71f94-119">Hvis du markerer feltet **Fast**, knytter du permanent varen til placeringen, hvilket betyder, at [!INCLUDE[d365fin](includes/d365fin_md.md)] altid vil forsøge at lægge den pågældende vare på placeringen, hvis der er plads til den, og at registreringen af tilknytningen bevares, selvom det aktuelle antal på placeringen er 0.</span><span class="sxs-lookup"><span data-stu-id="71f94-119">If you select the **Fixed** field, you are fixing the item to the bin, meaning that [!INCLUDE[d365fin](includes/d365fin_md.md)] will try to put this item in the bin if there is space for it, and it will preserve the record fixing the item to the bin even when the quantity in the bin is 0.</span></span> <span data-ttu-id="71f94-120">Der kan anbringes andre varer på placeringen, selvom en bestemt vare er fast tilknyttet placeringen.</span><span class="sxs-lookup"><span data-stu-id="71f94-120">Other items can be put into the bin, even though a particular item has been fixed to the bin.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="71f94-121">Du kan oprette flere placeringsindhold ad gangen på siden **Opr.kladde til placeringsindh.**.</span><span class="sxs-lookup"><span data-stu-id="71f94-121">You can set up several bin contents at the same time on the **Bin Content Creation Worksheet** page.</span></span>  

## <a name="to-create-bin-content-with-a-worksheet"></a><span data-ttu-id="71f94-122">Sådan oprettes placeringsindhold med en kladde</span><span class="sxs-lookup"><span data-stu-id="71f94-122">To create bin content with a worksheet</span></span>  
<span data-ttu-id="71f94-123">Når du har oprettet placeringer, kan du angive, hvilke varer der skal opbevares på hver placering vha. en kladde.</span><span class="sxs-lookup"><span data-stu-id="71f94-123">When you have created your bins, you can create the bin content that you want for each bin in the bin content creation worksheet.</span></span>

1.  <span data-ttu-id="71f94-124">Vælg ikonet ![Elpære, der åbner funktionen Fortæl mig](media/ui-search/search_small.png "Fortæl mig, hvad du vil foretage dig"), angiv **Opr.kladde til placeringsindh.**, og vælg derefter det relaterede link.</span><span class="sxs-lookup"><span data-stu-id="71f94-124">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Bin Content Creation Worksheet**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="71f94-125">Klik i feltet **Navn** i kladdehovedet, og vælg kladden for den lokation, hvor du vil oprette placeringsindhold.</span><span class="sxs-lookup"><span data-stu-id="71f94-125">On the worksheet header, in the **Name** field, select the worksheet of the location where you want to create bin contents.</span></span>  
3.  <span data-ttu-id="71f94-126">Vælg koden for den placering, hvis indhold du vil definere, i feltet **Placeringskode**.</span><span class="sxs-lookup"><span data-stu-id="71f94-126">In the **Bin Code** field, select the code of the bin for which you want to define bin content.</span></span>   

    <span data-ttu-id="71f94-127">Hvis du bruger styret læg-på-lager og pluk på denne lokation, udfyldes de felter, der vedrører den pågældende placering, automatisk, f.eks. **Placeringstype**, **Lagerklassekode** og **Placeringsniveau**.</span><span class="sxs-lookup"><span data-stu-id="71f94-127">If you are using directed put-away and pick in this location, the fields relating to that particular bin, such as **Bin Type**, **Warehouse Class Code**, and **Bin Ranking**, will be filled in automatically.</span></span> <span data-ttu-id="71f94-128">Det er oplysninger, du skal overveje, når du definerer placeringsindholdet.</span><span class="sxs-lookup"><span data-stu-id="71f94-128">This is information that you need to consider as you define the bin content.</span></span>  
4.  <span data-ttu-id="71f94-129">Vælg den vare, der skal knyttes til placeringen, og udfyld felterne vedrørende placeringsindhold.</span><span class="sxs-lookup"><span data-stu-id="71f94-129">Select the item that you want to assign to the bin, and fill in the fields related to the bin content.</span></span> <span data-ttu-id="71f94-130">Udfyld felterne **Maks. antal** og **Min. antal**, hvis du bruger styret læg-på-lager og pluk, og du planlægger at bruge funktionen **Beregn placeringsgenopfyldning**.</span><span class="sxs-lookup"><span data-stu-id="71f94-130">If you are using directed put-away and pick, and you want to use the **Calculate Bin Replenishment** function, fill in the **Max. Qty.** and **Min. Qty.** fields.</span></span>  

    <span data-ttu-id="71f94-131">Hvis du vil angive denne placering som den foretrukne placering for varen, selvom det placeringsantallet er **0**, og alle andre læg-på-lager-kriterier er de samme, skal du vælge feltet **Fast**.</span><span class="sxs-lookup"><span data-stu-id="71f94-131">To set this bin as the preferred bin for the item even if the bin quantity is **0** and all other put-away criteria equal, select the **Fixed** field.</span></span>  
5.  <span data-ttu-id="71f94-132">Gentag trin 3-4 for hver vare, du vil tildele til en placering.</span><span class="sxs-lookup"><span data-stu-id="71f94-132">Repeat steps 3 through 4 for each item you want to assign to a bin.</span></span>  
6.  <span data-ttu-id="71f94-133">Vælg handlingen **Udskriv** for at få vist og udskrive det placeringsindhold, du har angivet i kladden.</span><span class="sxs-lookup"><span data-stu-id="71f94-133">Choose the **Print** action to preview and print the bin content you have entered in the worksheet.</span></span> <span data-ttu-id="71f94-134">Fortsæt med at redigere placeringsindholdet, indtil du er tilfreds.</span><span class="sxs-lookup"><span data-stu-id="71f94-134">Continue to revise the bin content until you are satisfied.</span></span>  
7.  <span data-ttu-id="71f94-135">Når du er klar, skal du vælge handlingen **Opret placeringsindhold**.</span><span class="sxs-lookup"><span data-stu-id="71f94-135">When you are ready, choose the **Create Bin Content** action.</span></span>  

<span data-ttu-id="71f94-136">I denne kladde kan du arbejde med placeringsindholdslinjer for flere placeringer ad gangen og på den måde få et godt overblik over, hvilke varer der lægges på de forskellige placeringer i en given zone, række eller hylde.</span><span class="sxs-lookup"><span data-stu-id="71f94-136">In this worksheet, you can work with a number of bin content lines for a number of bins and thereby obtain a good overview of what you are putting into various bins in a given zone, aisle, or rack.</span></span>
