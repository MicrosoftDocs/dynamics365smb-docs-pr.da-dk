## <a name="assign-the-template-to-an-api"></a><span data-ttu-id="8f1df-113">Tildele skabelonen til et API</span><span class="sxs-lookup"><span data-stu-id="8f1df-113">Assign the template to an API</span></span>

2. <span data-ttu-id="8f1df-116">Vælg **Ny**, og vælg derefter **Rækkefølge**-værdien for posten.</span><span class="sxs-lookup"><span data-stu-id="8f1df-116">Choose **New**, and then choose the **Order** value for the record.</span></span>  
<span data-ttu-id="8f1df-117">Hvis der er valgt mere end én skabelon for et API (side-id), anvendes skabelonerne i den rækkefølge, der er defineret i kolonnen **Rækkefølge**.</span><span class="sxs-lookup"><span data-stu-id="8f1df-117">If there is more than one template selected for an API (Page ID), the templates are applied in the order defined in the **Order** column.</span></span>   
<span data-ttu-id="8f1df-118">Når hver skabelon anvendes, anvendes feltværdier, der er defineret i skabelonen, kun til felter, der ikke allerede har en defineret værdi, enten direkte i API'et eller i en tidligere anvendt skabelon i rækkefølgen.</span><span class="sxs-lookup"><span data-stu-id="8f1df-118">When each template is applied, field values defined in the template are only applied to fields that have not already had a value defined, either explicitly in the API, or in a previously applied template in the order.</span></span> 
3. <span data-ttu-id="8f1df-119">Vælg en værdi for **Side-id**:</span><span class="sxs-lookup"><span data-stu-id="8f1df-119">Select a **Page ID** value.</span></span>  
<span data-ttu-id="8f1df-120">Dette er siden for det API, som skabelonen skal anvendes på.</span><span class="sxs-lookup"><span data-stu-id="8f1df-120">This is the page for the API to which the template will be applied.</span></span> <span data-ttu-id="8f1df-121">**Side-ID**-opslaget indeholder en oversigt over alle API'er, der er tilgængelige i biblioteket.</span><span class="sxs-lookup"><span data-stu-id="8f1df-121">The **Page ID** lookup provides a list of all APIs available in the library.</span></span>
4. <span data-ttu-id="8f1df-122">Vælg en værdi i feltet **Skabelonkode**.</span><span class="sxs-lookup"><span data-stu-id="8f1df-122">Select a value in the **Template Code** field.</span></span>  
<span data-ttu-id="8f1df-123">Skabelonkoden er koden for den skabelon, der blev defineret på siden **Konfigurationsskabeloner**.</span><span class="sxs-lookup"><span data-stu-id="8f1df-123">The template code is the code for the template that was defined on the **Configuration Templates** page.</span></span> <span data-ttu-id="8f1df-124">De definerede skabelonværdier anvendes på API'et.</span><span class="sxs-lookup"><span data-stu-id="8f1df-124">The template values defined are applied to the API.</span></span> 
5. <span data-ttu-id="8f1df-125">I feltet **Betingelser** skal du angive, hvilken skabelon der skal anvendes.</span><span class="sxs-lookup"><span data-stu-id="8f1df-125">In the **Conditions** field, specify which template should be applied.</span></span>  
<span data-ttu-id="8f1df-126">Den definerede skabelon anvendes på en ny post, der oprettes via API'et, hvis og kun hvis de betingelser, der er defineret i feltet **Betingelser**, opfyldes af de værdier, der allerede er defineret for den nye forekomst af enheden.</span><span class="sxs-lookup"><span data-stu-id="8f1df-126">The defined template is applied to a new record created through the API if, and only if, the conditions defined in the **Conditions** field are met by the values already defined for the new instance of the entity.</span></span>
